# Installing Git Subdirectories with pnpm

Til you can [install packages from subdirectories of git repositories](https://pnpm.io/package-sources#install-from-a-subdirectory-of-a-git-repository) using pnpm. 

```bash
pnpm add RexSkz/test-git-subfolder-fetch#path:/packages/simple-react-app
```
